Política de Privacidade

Compromisso com a segurança, transparência e proteção dos seus dados pessoais, conforme a Lei Geral de Proteção de Dados (LGPD).

A J L Serviços Contábeis respeita a sua privacidade e está comprometida com a proteção dos dados pessoais coletados por meio deste site. Esta Política de Privacidade explica como os dados são coletados, utilizados, armazenados e protegidos.

Base Legal

Esta Política de Privacidade foi elaborada em conformidade com a Lei nº 13.709/2018 — Lei Geral de Proteção de Dados Pessoais (LGPD), que regula o tratamento de dados pessoais no Brasil.

Quais dados coletamos

Podemos coletar os seguintes dados pessoais:

  • Nome completo
  • Telefone / WhatsApp
  • E-mail
  • CPF (quando necessário para prestação do serviço)
  • Informações fornecidas voluntariamente em formulários

Finalidade do uso dos dados

Os dados coletados são utilizados para:

  • Entrar em contato com o cliente
  • Prestar os serviços contábeis contratados
  • Emitir documentos fiscais e obrigações legais
  • Cumprir exigências legais e regulatórias

Compartilhamento de dados

Seus dados pessoais não são vendidos ou compartilhados com terceiros, exceto quando necessário para cumprimento de obrigações legais, fiscais ou regulatórias.

Direitos do titular dos dados

Nos termos do artigo 18 da LGPD, o titular dos dados pessoais possui os seguintes direitos:

  • Confirmar a existência do tratamento
  • Acessar os dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão dos dados, quando aplicável
  • Revogar o consentimento

Segurança dos dados

Adotamos medidas técnicas e organizacionais adequadas para proteger os dados pessoais contra acessos não autorizados, perdas ou vazamentos.
